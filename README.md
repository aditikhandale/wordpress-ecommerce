# WordPress E-Commerce Website

This project is a fully functional E-Commerce website built using WordPress and WooCommerce.

## 🔧 Features
- Product Listing Page
- Product Details Page
- Add to Cart
- Checkout System
- User Login & Registration
- Order Confirmation
